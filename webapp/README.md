# 🛒 Telegram Web App — Магазин аккаунтов Facebook

## Описание
Полноценный интернет-магазин внутри Telegram для продажи аккаунтов Facebook. Оплата криптовалютой, автовыдача после оплаты.

## Структура проекта
```
tg-webapp-shop/
├── index.html          # Главный файл (HTML + Web App API)
├── css/
│   └── style.css       # Стили (тёмная тема, адаптивный дизайн)
├── js/
│   └── app.js          # Логика приложения
└── images/             # Изображения (пустая папка)
```

## Функционал

### ✅ Реализовано во frontend:
- [x] Главный экран с категориями товаров
- [x] Каталог с фильтрами (страна, возраст, тип)
- [x] Карточки товаров с детальной информацией
- [x] Корзина с добавлением/удалением товаров
- [x] Оформление заказа с подсчётом суммы
- [x] Интеграция Telegram Web App API (цвета, кнопки, haptic)
- [x] История заказов с выдачей данных аккаунтов
- [x] Адаптивный дизайн под мобильные устройства
- [x] Сохранение корзины и заказов в localStorage

### 🔜 Требует backend:
- [ ] Интеграция CryptoBot API (создание счетов, проверка оплаты)
- [ ] База данных товаров (PostgreSQL/SQLite)
- [ ] Автоматическая выдача аккаунтов после оплаты
- [ ] Админ-панель для добавления товаров
- [ ] Защита от повторной продажи одного аккаунта

## Как запустить

### 1. Загрузка на сервер
```bash
# Скопируйте файлы на ваш сервер (VPS)
scp -r tg-webapp-shop/* root@your-server-ip:/var/www/html/
```

### 2. Настройка Telegram Bot
1. Создайте бота через [@BotFather](https://t.me/BotFather)
2. Получите токен бота
3. Включите Web App:
   - `/mybots` → выберите бота → Bot Settings → Menu Button → Configure menu button
   - Укажите URL вашего Web App (например: `https://your-domain.com`)

### 3. Настройка Web App URL
В `index.html` убедитесь, что скрипт Telegram загружается:
```html
<script src="https://telegram.org/js/telegram-web-app.js"></script>
```

### 4. Для тестирования локально
Откройте `index.html` в браузере. Для полной функциональности открывайте через Telegram.

## Следующий шаг — Backend
Для полноценной работы нужен Python-бот с:
- aiogram 3.x для обработки команд
- CryptoBot API для приёма оплаты
- SQLite/PostgreSQL для хранения аккаунтов и заказов
- Автовыдачей после подтверждения оплаты

## Технологии
- **Frontend**: HTML5, CSS3, Vanilla JS, Telegram Web App API
- **Backend (планируется)**: Python, aiogram, FastAPI
- **Платежи**: CryptoBot API
- **База данных**: SQLite → PostgreSQL
